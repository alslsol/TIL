# 마크다운 기본 문법
## 1. 제목

1-4개의 # 기호를 사용해 제목 수준을 지정

## 2. 텍스트 스타일
- 굵기: **단어**
- 기울기: *단어*
- 취소선: ~~단어~~
- 굵기+기울기: ***단어***

## 3. 텍스트 인용
>인용구문

## 4. 코드 인용(핵심)
- in-line에서 코드 인용: 이것은 `code` 입니다.
- 코드 자체 인용할 때: 백틱(`) 활용

- 코드 블럭: 백틱(`) 3개로 열고 닫기
    -첫 백틱 옆에 프로그램명 적으면, 각 코드 색으로 변경

```python
def hello():
    print("hello!")
```
